# Application Commands Guide

Your AI assistant can now open applications and websites on your computer!

## How to Use

Simply ask Brain to open an application or website using natural language:

### Opening Applications

**Examples:**
- "Open notepad"
- "Launch calculator"
- "Start Chrome"
- "Open Visual Studio Code"
- "Run Spotify"
- "Open File Explorer"

### Supported Applications

The assistant can open these common applications:

- **Text Editors**: Notepad, Visual Studio Code
- **Office**: Word, Excel, PowerPoint
- **Browsers**: Chrome, Firefox, Edge
- **System**: Calculator, Paint, File Explorer, Command Prompt, PowerShell, Task Manager, Settings, Control Panel
- **Media**: Spotify
- **Gaming**: Steam, Discord
- **Design**: Photoshop

### Opening Websites

**Examples:**
- "Open google.com"
- "Go to youtube.com"
- "Visit github.com"
- "Open https://www.example.com"

The assistant will automatically open the website in your default browser.

### How It Works

1. You type a command like "Open notepad"
2. Brain detects it's a system command
3. The application/website opens automatically
4. Brain confirms the action

### Security

- Only safe, predefined commands are executed
- System commands are logged for security
- No dangerous operations are allowed

### Troubleshooting

If an application doesn't open:
- Make sure the application is installed
- Check if the application name is spelled correctly
- Try using the full application name (e.g., "Visual Studio Code" instead of "VSCode")
